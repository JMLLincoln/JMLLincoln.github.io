JavaScript Junk

The first game 'I' programmed in JavaScript was snake. I saw a 'snake in 4 minutes 30 seconds' video which I attempted to copy from memory.
Here's a link to the video.

This inspired me to make my own games using only HTML/CSS and JavaScript and this is what I came up with:

  • Snake

    'Up Down Left Right' / 'WASD' to control the snake. Eat the monotone apples to score points and increase your size. Don't eat yourself idiot.

    Play!

  • Falling Down

    The roof is lava okay. Don't touch it or you'll d. Try to stay as close to the bottom as possible to survive and rack up them dere points. 'Left Right' / 'A D'to move.

    Play!

  • Memory Card Game

    Select two cards and try to get a match. If you're not successful, memorise the card positions as best you can. Collect more cards in total than your opponent.

    Play!

  • Sand Piles

    The idea is pretty straightforward: place sand in a cell and if there are four or more, they topple and spread to the neighbouring cells.

    Play!

  • Shooter

    'Up Down Left Right' / 'WASD' to control the block. Shoots towards the cursor. Touching the hollow blocks will hurt you. Gain points for surving and shooting hollows.

    Play!

  • Breakout

    The aim is to break as many block as possible by bouncing the white block towards the coloured blocks. 'Left Right' / 'A D' to move. You're vertically disabled so don't drop it.

    Play!

  • Battle Shooters

    Same controls as 'Shooter'. Added 'rooms' to act as levels. Each level is completed when all enemies are defeated then doors open up to the next level.

    Play!
